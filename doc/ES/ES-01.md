# ES 1장 훑어보기 

#### ES 특징 

- 준 실시간성 
  => 데이터 입력 시, 메모리에 저장되며, 일정 시간 후 샤드에 저장된다.
- Cluster 를 이용해 안정적인 운영 
  => 여러개의 노드로 구성되며, 데이터를 노드에 분산하여 저장한다. 
  => `keyword` primary shard / replica shard / 안정성
- RestApi 
  => restApi 로 요청하고 json 형태를 사용한다.
- schemeless
  => 동적으로 scheme 를 생성한다. 
  => `keyword` 동적 생성 / 정적 생성
